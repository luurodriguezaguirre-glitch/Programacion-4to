# Programación - 4to Año

Bienvenidos/as a la materia de Programación. Durante el año vamos a aprender los fundamentos de **Python** y, con esas herramientas, construir un **videojuego** propio.

## 🎯 Objetivos de la materia

- Comprender la lógica de programación y adquirir el pensamiento computacional.
- Aprender a leer, escribir y depurar código en Python.
- Desarrollar un proyecto final integrador: un videojuego.

## 🗂️ Contenidos

### Unidad 1 — Introducción a Python
- ¿Qué es programar? Instalación y entorno de trabajo.
- Variables y tipos de datos (int, float, str, bool).
- Entrada y salida de datos (`input()`, `print()`).
- Operadores aritméticos, relacionales y lógicos.

### Unidad 2 — Estructuras de control
- Condicionales: `if`, `elif`, `else`.
- Bucles: `while` y `for`.
- Uso de `range()`, `break` y `continue`.

### Unidad 3 — Estructuras de datos
- Listas: creación, indexado, métodos y recorrido.
- Tuplas.
- Diccionarios: claves, valores, métodos y recorrido.

### Unidad 4 — Funciones
- Definición de funciones (`def`).
- Parámetros, argumentos y retorno de valores.
- Alcance de variables (scope).

### Unidad 5 — Proyecto final: Videojuego 🎮
- Diseño del juego: idea, mecánicas y reglas.
- Aplicación de todo lo aprendido (variables, condicionales, bucles, listas, diccionarios y funciones).
- Uso de una librería para gráficos (por ejemplo, `pygame`).
- Presentación y entrega del proyecto.

## 🧰 Requisitos

- Python 3.10 o superior instalado.
- Editor de código (recomendado: VS Code).
- Librería `pygame` para el proyecto final (`pip install pygame`).

## 📅 Modalidad de trabajo

- Clases teórico-prácticas con ejercicios en cada encuentro.
- Prácticas integradoras al finalizar cada unidad.
- Proyecto final grupal o individual, a definir.

## ✅ Evaluación

- Ejercicios y entregas parciales durante el año.
- Proyecto final (videojuego) con defensa oral.
- Participación y proceso de trabajo en clase.
